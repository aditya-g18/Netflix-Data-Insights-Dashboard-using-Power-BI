# 🎬 Netflix Content Analytics Dashboard (Power BI)

## 📌 Project Overview

This project presents an interactive dashboard built using Power BI to analyze Netflix content data. It provides insights into movies and TV shows, helping users understand content trends, distribution, and key metrics.

---

## 📊 Key Features

* Total Titles, Movies, and TV Shows KPI cards
* Country-wise content distribution (Top 10 countries)
* Movie vs TV Show comparison
* Rating distribution analysis
* Content trends over release years
* Interactive filters (Genre, Release Year)

---

## 📁 Dataset

* Source: Netflix dataset (CSV format)
* Contains information like:

  * Title
  * Type (Movie / TV Show)
  * Country
  * Release Year
  * Rating
  * Genre

---

## 🛠️ Tools & Technologies

* Power BI
* Power Query (Data Cleaning & Transformation)
* DAX (Measures & Calculations)

---

## 📈 Key Insights

* Movies dominate the platform compared to TV Shows
* The United States has the highest content count
* Most content is concentrated in recent years
* TV-MA and TV-14 are the most common ratings

---

## 🎨 Dashboard Highlights

* Clean dark theme inspired by Netflix UI
* Use of brand color (#E50914)
* Interactive visuals and slicers for better exploration

---

## 🚀 How to Use

1. Open the `.pbix` file in Power BI Desktop
2. Use slicers to filter data (genre, year, etc.)
3. Explore visuals for insights

---

## 📌 Conclusion

This dashboard helps in understanding Netflix content strategy and distribution patterns using data visualization.
